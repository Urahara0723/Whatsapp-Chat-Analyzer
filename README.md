# WhatsApp Chat Analyzer

WhatsApp Chat Analyzer built using Python and Streamlit that provides deep insights into personal and group conversations through data analysis and interactive visualizations.

## Features

### Chat Analytics

* Total messages
* Total words
* Media shared
* Links shared

### Timeline Analysis

* Monthly activity timeline
* Most active day analysis
* Most active time analysis

### Group Analysis

* Most active users
* User contribution percentages
* Interaction insights

### NLP Features

* Most common words
* Hinglish stop-word removal
* Emoji analysis
* Shared interests detector

### AI & Behavioral Features

* Chat personality analyzer
* Conversation starter analysis

### Visualizations

* Treemaps
* Activity graphs
* Network graphs
* Bar charts

# Installation

Clone the repository:

```bash
git clone https://github.com/Urahara0723/Whatsapp-Chat-Analyzer.git
```

Move into the project folder:

```bash
cd Whatsapp-Chat-Analyzer
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```

# How to Use

1. Export your WhatsApp chat as ZIP.
2. Upload the ZIP file into the app.
3. Explore analytics, insights, and visualizations.
